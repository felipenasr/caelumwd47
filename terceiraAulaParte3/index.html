Module Pattern

O grande segredo para criar um module pattern eficiente é:

  • Começamos criando uma função anonima. Não podemos deixar a função anonima solta no código. Ela sempre tem que estar acompanhada: de uma função, uma variável ou ser tratada como parametro. Para função anonima rodar no nosso código, temos que coloca-la entre paraenteses:

    (function(){});

  • Quando a página carrega, queremos que a função seja executada. Para isso temos que executar a função:

    (function(){})();

    Esse tipo de sintaxe é chamado de I.I.F.E, em portugues Função Imediata
