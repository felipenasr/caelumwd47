Funções

Existe um padrão de escrita de funções dentro do universo JavaScript

O primeiro padrão é o expressão de função nomeada. Este é o padrão mais simples e o mais comum dentro do universo JS

function saudacao(){
   var msg = "Boa Noite";
   console.log(msg);
}

O segundo padrão é a Expressão de Função Anonima.

Quando se trabalha com uma função anomina, obrigatoriamente devemos atribuila a um valor dentro do nosso código

var mensagem = function () {
   const msg = 'Boa Noite!';
   console.log(msg);
}

Macetes da linguagem

Podemos utilizar o conector lógico OR -> ( || ), para fazermos uma 'validação' dentro do código, sem que precisar utilizar IF/ELSE. Exemplo:

let soma = function (a) {
   return a || 0;
}
