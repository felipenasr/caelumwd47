Objetos

JavaScript é uma linguagem extremamente ligado a orientação ao objeto.


Conceito de orientação ao Objeto:

Um objeto é composto por ações e características

As características(propriedades/atributos) são aquilo que compões um objeto.

As ações(métodos) são tudo aqui que um objeto pode fazer. Exemplo:

Carro:

Cor é uma características(propriedades/atributos)

Acelerar é uma ação(métodos)

Voltando aos objetos em JS

Dot Notation

Para setar um valor utilizando o Dot Notation precisamos utilizar o ponto após o nome do objeto. Exemplo:

var carros = {};
carros.modelo = 'Fox';

Bracket Notation

Para setar um valor utilizando o Braket Notation, utilizamos o nome do objeto seguido pela propriedade dentro de colchetes. Exemplo

var carros = {};
carros["cor"] = 'Azul';
